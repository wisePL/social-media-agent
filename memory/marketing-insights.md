## 📈 검증된 고성과 카피 패턴

### 패턴 1: 극단적으로 짧은 CTA
```
"You can build it for free." → 5.77% eng rate
"Don't tap." → 12K+ imp, 5%+ eng
"POV: vibe coding on Verse8" → 18K imp
"Traditional game dev didn't make it. Vibecoders are thriving." → 13K imp
```
**룰**: 1-2줄 + 링크/영상. 설명 없음.

### 패턴 2: 대비/반전 (Before→After)
```
"Manual coding vs vibe coding. Guess which one shipped faster."
"Studios spend months. On Verse8, it starts in minutes."
"4 years of college or 5 minutes on Verse8?"
"Most people think making a game takes years. On Verse8, it takes a prompt."
```
**룰**: 구체적 시간/비용 대비. "On Verse8" 브랜드 삽입.

### 패턴 3: 도발적 언어 (Degen Tone)
```
"Making a game is so easy now. Steal this game idea." → 25,974 imp, 1.9% eng
"Skill issue or colorblind?" → 21,973 imp
"Even your mom can join" (Grand Game Jam)
"This is the most @waleswoosh way to hire a reply guy." → 19,786 imp
```
**룰**: 커뮤니티 언어. 살짝 도발적. 게임 링크와 함께.

### 패턴 4: 구체적 수치 (Social Proof)
```
"Wall Smash: 193K plays and counting. A Verse8 creator built it in hours."
"25,000+ games generated. 4,000+ creators. 3.5M+ monthly active users."
"While traditional devs spend months... Verse8 creators go from idea → live game in minutes."
```
**룰**: 플랫폼 성장 지표 + 크리에이터 스토리. 숫자 구체적으로.

### 패턴 5: 게임 스포트라이트 (Creator POV)

## 🔍 A/B 분석 인사이트 (3개월 데이터 기반)

### 훅 패턴별 평균 성과

| 훅 유형 | 평균 Eng Rate | 평균 임프레션 | 권장 용도 |
|---------|-------------|------------|---------|
| **수치형** | ~2.5% | 20K+ | 프로그램 발표, 마일스톤 |
| **반전형** | ~2.0% | 20K+ | 포지셔닝, 비교 |
| **금지/도발형** | ~1.9% | 20K+ | 게임 데모, 커뮤니티 |
| **진술형** | ~1.6% | 15K+ | 브랜드, 제품 업데이트 |
| **질문형** | ~1.5% | 12K+ | 참여 유도, AMA |

**결론**: 수치형 > 반전형 > 도발형 순으로 효과적. 질문형은 리플 수는 많지만 임프레션 낮음.

### 프레임워크별 성과

| 프레임워크 | 적합한 유형 | 성과 |
|-----------|------------|------|
| **BAB** (Before→After→Bridge) | 제품 업데이트, 포지셔닝 | 최고 (1.5%~2.0%) |
| **PAS** (Problem→Agitate→Solution) | 파트너십, 커뮤니티 페인 | 좋음 (1.5%~2.5%) |
| **4Us** (Useful+Urgent+Unique+Ultra-specific) | 이벤트, Game Jam | 이벤트에서 최고 (3%~6%) |
| **극단 축약** (1-2줄 + 링크) | 데모, CTA | 임프레션 낮지만 eng rate 최고 (5%+) |

### 이미지/영상 효과

| 포맷 | 평균 임프레션 | 비고 |
|------|------------|-----|
| 영상 (게임플레이/데모) | 20K~79K | 임프레션 압도적으로 높음 |
| 이미지 (인포그래픽) | 15K~30K | 중간 |
| 텍스트만 | 10K~20K | 훅 퀄리티에 100% 의존 |

**결론**: 게임플레이 영상 > 인포그래픽 이미지 > 텍스트 순. 그러나 eng rate는 반전될 수 있음.
